## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npx expo start
   ```

3. Run on your device:
   - Scan the QR code with Expo Go app
   - Or press 'i' for iOS simulator
   - Or press 'a' for Android emulator
